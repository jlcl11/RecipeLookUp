  ## About                                                                                                                                                                                                                                                                   
                                                                                                                                                                                                                                                                             
  RecipeLookUp brings recipe browsing into spatial computing. Built entirely with SwiftUI for visionOS, it explores how everyday tasks like cooking can benefit from the Apple Vision Pro's immersive environment.                                                         
                                          
  Recipes are fetched from [TheMealDB](https://www.themealdb.com/) API and can be saved locally with SwiftData.
                                                                                                                                                                                                                                                                             
  ## Features
                                                                                                                                                                                                                                                                             
  - Browse and search recipes from an extensive database                                                                                                                                                                                                                   
  - Save favorites locally using SwiftData                                                                                                                                                                                                                                   
  - Designed for the spatial computing experience on Apple Vision Pro                                                                                                                                                                                                      
  - Native SwiftUI interface adapted to visionOS interaction patterns
                                              
  ## Architecture                                                                                                                                                                                                                                                            
  
  - **SwiftUI** for declarative UI on visionOS                                                                                                                                                                                                                               
  - **SwiftData** for local persistence                                                                                                                                                                                                                                    
  - **async/await** for network calls                                                                                                                                                                                                                                        
  - **MVVM** pattern for state management                                                                                                                                                                                                                                    
                                              
  ## Tech Stack                                                                                                                                                                                                                                                              
                                                                                                                                                                                                                                                                           
  `SwiftUI` · `visionOS` · `SwiftData` · `REST API` · `MVVM` · `Swift Concurrency`                                                                                                                                                                                           
                                          
  ## Requirements                                                                                                                                                                                                                                                            
                                                                                                                                                                                                                                                                             
  - Xcode 15.0+                               
  - visionOS 1.0+ (Apple Vision Pro simulator or device)
